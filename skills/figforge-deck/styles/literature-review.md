# Style: literature-review (文献综述 / 综述汇报)

Target: a survey or review talk that emphasizes structure, lineage, and breadth of
references. Visual language reads "library / archive / index card" rather than slide deck.

---

## Anchors (constant across the deck)

1. **Color palette**
   - base: parchment `#EFE8D8`
   - secondary: ink `#1F1B16`
   - accent: ochre `#B8862C`

2. **Typography**
   - display: a transitional serif (think Crimson Pro / Lora / EB Garamond)
   - body: a slab serif or monospace serif (think IBM Plex Serif / IBM Plex Mono)
   - never use sans for the display

3. **Layout grid**
   - cover → centered "journal cover" composition
   - section → full-bleed timeline ribbon mid-slide
   - content → 3-column "index card" grid
   - closing → centered with hairlines above and below

4. **Recurring motif**
   - a small filled circle ● followed by a hairline (the "reference marker"), placed
     top-right of every slide, same size, same offset

5. **Background type**
   - parchment paper texture (subtle low-contrast noise, never glossy)

---

## Per-role layout

### cover

- Background: parchment with subtle noise
- Ochre rule above the title (long, full text-column width)
- Title: large transitional serif, ink color, centered
- Subtitle (≤1 caption): small slab serif, centered, just below the title
- The composition should read like the cover of a printed journal

### section

- Background: parchment
- A full-bleed horizontal "timeline ribbon" mid-slide: 3–5 abstract era markers spaced
  along a hairline, no real dates baked in unless the caption supplies them
- Section title: large transitional serif, ink, above the ribbon
- Optional 1-line caption: slab serif, beneath the ribbon
- Top-right reference marker as defined

### content

- Background: parchment
- Slide title: transitional serif, ink, spans the top of the slide (single line)
- Below the title: a 3-column grid of "index cards" with a 1-px ink outline and a
  parchment fill, each card the same size
- If captions are provided, they appear AS column subheaders (caption 1 = column 1
  header, caption 2 = column 2 header). Card bodies stay decorative — short abstract
  marks, never fake citation text or fake bullet rows
- Top-right reference marker

### closing

- Background: parchment
- Single centered serif word: "Open Questions" / "Discussion" / "References"
- Ochre hairline above and below the word
- Top-right reference marker

---

## Visual element voice

- Prefer schematic timelines, taxonomic trees, citation-network silhouettes (abstract,
  no real reference text), comparison matrices
- Prefer index-card / archive-card / library-card aesthetic
- Avoid: 3D, photographs, glow, neon, gradient blobs, social-media-style icons
- Treat the slide like a page from a printed book or journal

---

## What NOT to do

- No bullet lists with check marks or arrows
- No gradient backgrounds
- No drop shadows on cards (a flat 1-px outline is the only allowed border)
- No accent line directly under the title
- No three-icon corporate grid (the index-card grid is the only grid here)
- No emoji
